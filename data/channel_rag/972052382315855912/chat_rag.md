# VRC世界旅行 基本情報

VRC世界旅行は、VRChat上で世界各地の観光地を巡る旅行体験イベント。
現実に近い体験（Virtual ≒ Real）をコンセプトとし、スタッフによるガイド付きツアー形式で進行する。

開始日は 2022年04月09日。
初回は「第0回 イタリア編」。


# 体験内容

- 世界中の実在観光地を3D空間で巡る
- スタッフによる現地解説・ガイドあり
- 飛行機・電車など移動演出あり
- 修学旅行のようなツアー体験
- ニッチなワールドや未発見スポットも紹介


# 特徴

- 没入感を重視したリアル寄りの体験設計
- チケット・招待状・パンフレットなど旅行演出あり
- イベントごとに構成・訪問地が変化
- スタッフ同行型の進行（自由探索ではない）

# 開催・運営情報

- 主催：NEKO旅
- スタッフ数：約13〜25人（時期により変動）
- 開催頻度：不定期（主に月末）
- 参加形式：事前予約制
- プラットフォーム：VRChat（主にPC）
- モバイル対応版：VRC世界旅行EXあり


# 参加方法

1. X(Twitter)（@VRC_WorldTour）をフォロー
2. 告知リンクから予約フォーム入力
3. 当選者に通知
4. スタッフにフレンド申請して参加


# ロゴの意味

- 帽子 = スタッフ（ガイド・引率）の象徴
