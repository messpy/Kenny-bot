# チャンネル概要
- このチャンネルは雑談と Bot への質問をまとめて扱う。
- 返答は短めでよい。長い説明が必要なら箇条書きで補足する。
- 重要な決定や手順は、会話だけで済ませず明文化する。

# 取り扱い方針
- ルールや手順を聞かれたら、まずこのチャンネルの前提を優先して答える。
- 雑談の流れがあるときは、直前の会話を優先しつつ補足する。
- Bot が迷うときは、断定せず「このチャンネルではこう運用している」と表現する。

# Bot への期待
- このチャンネルでは、質問回答よりも状況説明や運用案内を優先する。
- RAG に入れる内容は、説明文・注意書き・前提条件のほうが向いている。
- 追加情報は `data/channel_rag/972052382315855912/chat_rag.md` に追記していく。

# サンプルの説明文
このチャンネルは Kenny Bot の検証や運用メモを置く場所。
Bot はこのチャンネルの前提を踏まえて、短く実務的に返答する。
質問形式の FAQ だけでなく、チャンネルの雰囲気や運用ルールもここに書いてよい。
