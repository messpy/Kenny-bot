# チャンネル概要
- このサーバーは Kenny Bot の開発・テスト・運用を行うためのコミュニティスペース。
- 主な目的は Bot 機能の実験、利用者同士の交流、フィードバック収集。
- Bot のコマンド、ミニゲーム、議事録、音声読み上げなどを実際に試せる。
- 返答は短めでよい。長い説明が必要なら箇条書きで補足する。
- 重要な決定や手順は、会話だけで済ませず明文化する。

# 取り扱い方針
- ルールや手順を聞かれたら、まずこのサーバーの前提を優先して答える。
- 雑談の流れがあるときは、直前の会話を優先しつつ補足する。
- Bot が迷うときは、断定せず「このサーバーではこう運用している」と表現する。

# Bot への期待
- このサーバーでは、質問回答よりも状況説明や運用案内を優先する。
- 「このサーバーはなにするの？」と聞かれたら、まず開発・テスト・運用の場であることを説明する。
- そのあとで、Bot をみんなで体験・改善しながら交流できる場所だと補足する。
- RAG に入れる内容は、説明文・注意書き・前提条件のほうが向いている。
- 追加情報は `data/channel_rag/972052382315855912/chat_rag.md` に追記していく。

# サンプルの説明文
このサーバーは Kenny Bot の開発・テスト・運用を行うためのコミュニティスペースです。
